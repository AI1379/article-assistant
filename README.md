# Article Assistant

一个帮助你自动~~生产学术垃圾~~编写论文的小机器人

## Features

- [ ] 基础功能：以 Markdown 格式输出一篇看上去像模像样的学术垃圾
- [ ] 一个简单的 RAG 知识库，可以更针对性的生产学术垃圾
- [ ] MCP 支持，生产更高质量更现代化的学术垃圾
- [ ] 多级目录结构的生成，可以生产更长更有逻辑的学术垃圾
- [ ] 一组内置工具，包括：
  - [ ] 文件读写
  - [ ] 自动调用 `numpy` `scipy` `sympy` `matplotlib` 等库计算和生成图表
- [ ] 支持 LaTeX 和 `.docx` 的输出
- [ ] 一个简单的 Web API 以及对应的前端页面，也许可以套一层 `PyWebView` 封装

## 架构

采用了多 Agent 共同工作的模式。
